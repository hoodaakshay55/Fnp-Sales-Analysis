# 📊 Sales Analysis Dashboard (Excel)

## 📌 Project Overview

This project focuses on building an interactive **Sales Analysis Dashboard** using Excel to analyze business performance and generate actionable insights. The dashboard helps track key metrics such as revenue, orders, customer behavior, product performance, and delivery efficiency.

---

## 🎯 Objectives

* Analyze overall sales performance and revenue trends
* Identify top-performing products and categories
* Understand customer purchasing behavior
* Evaluate regional (city-wise) performance
* Provide insights for data-driven business decisions

---

## 📊 Key Insights

* 💰 Generated **₹35.2L+ revenue** from **1000 orders**
* 📅 Peak sales observed during **Anniversary, Raksha Bandhan, and Holi**
* 🛍 Top products: **Magnam Set, Quia Gift**
* ⏰ Highest order activity during **evening hours**
* 🌍 Top-performing cities include **Kavali, Imphal, and Dhanbad**
* 🚚 Average delivery time: **5.53 days**

---

## 📈 Dashboard Features

* Revenue analysis by **Occasion, Month, Category, and City**
* **Top 5 products** by revenue
* **Hourly order trend analysis**
* Interactive **filters/slicers** for dynamic exploration
* Clean and user-friendly dashboard design

---

## 🛠 Tools & Technologies

* Microsoft Excel

  * Pivot Tables
  * Charts & Graphs
  * Slicers & Filters
  * Dashboard Design

---

## 📂 Dataset

The dataset contains sales transaction details including:

* Order ID
* Product Name & Category
* Order Date & Delivery Date
* Revenue
* City & Occasion
